# VWC Skills

A collection of AI skills created as part of the Vets Who Code Skills as Curriculum coursework.

## Skills

### Git Workflow Assistant
Provides guidance for standard Git workflows including branching, commits, pull requests, and repository synchronization.

### README Documentation Writer
Generates clear and professional README documentation for software projects.

### Project Planning Assistant
Transforms project ideas into structured plans, milestones, feature lists, and implementation roadmaps.
