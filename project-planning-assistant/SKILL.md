# Project Planning Assistant

## <ins>Purpose</ins>

Effective project planning is crucial for the success of any project. This skill helps users organize their projects, set clear goals, define tasks, and create a roadmap to ensure timely completion and efficient resource management.

## <ins>Procedure</ins>

1. Define the project goal and desired outcomes:
- Define the overall purpose of the project.
- Set specific, measurable, achievable, relevant, and time-bound (SMART) goals.

2. Identify the primary users and document their needs::
- Identify the primary audience for the project.
- Understand their needs and preferences to tailor the project accordingly.

3. List major features and functionalities:
- Brainstorm and document the key features that the project will include.
- Prioritize features based on their importance and feasibility.

4. Break features into smaller tasks:
- Decompose each feature into smaller, manageable tasks.
- Assign tasks to team members if applicable.

5. Prioritize tasks:
- Use a prioritization method (e.g., Eisenhower Matrix, Eat That Frog-ABCDE method) to determine the order of priority for tasks.
- Focus on high-priority tasks that contribute significantly to project goals.

6. Establish measurable milestones for each project phase:
- Set milestones to track progress and ensure timely completion of key phases of the project.
- Use milestones to celebrate achievements and maintain motivation.

7. Recommend implementation order:
- Suggest an optimal order for implementing features and tasks based on dependencies and priorities.
- Consider starting with foundational features that other tasks depend on.

## <ins>Examples</ins>

### Example Input

"I want to build an accountability app that helps users track their daily habits and stay motivated called AccountaBoss.

### Example Output

1. Define project goals and objectives:
- Goal: Create an accountability app that helps users track their daily habits and stay motivated.
- Objectives:
  - Allow users to create and track daily habits.
  - Provide reminders and notifications to keep users on track.
  - Include a social feature for users to share their progress and encourage each other.

2. Define target users:
- Individuals looking to improve their habits and stay accountable.

3. List major features and functionalities:
- Habit tracking
- Reminders and notifications
- Social sharing and community support

4. Break features into smaller pieces:
- Habit tracking:
  - Create habit
  - Edit habit
  - Mark habit as completed
  - Delete/deactivate/archive habit
- Reminders and notifications:
  - Set reminder time
  - Receive notifications for upcoming habits
- Social sharing and community support:
  - Share progress with friends
  - Join groups for accountability

5. Prioritize tasks:
- Start with habit tracking due to it being the core feature of the app.
- Follow with reminders and notifications to enhance user engagement.
- Finally, implement social sharing and community support to gain user engagement.

6. Create milestones:
- Milestone 1: Complete habit tracking feature (2 weeks)
- Milestone 2: Implement reminders and notifications (1 week)
- Milestone 3: Add social sharing and community support (2 weeks)

7. Recommend implementation order:
- Start with habit tracking to establish the core functionality of the app.
- Next, implement reminders and notifications to enhance user engagement and retention.
- Finally, add social sharing and community support to foster a sense of community and encourage users to stay accountable.
